# Objetivos del proyecto
Analizar el comportamiento de los clientes de ConectaTel, para así poder identificar los segmentos, patrones de uso, factores que esten asociados al churn, con la finalidad de generar acciones estrategicas que refuerzen y optimicen la oferta de planes para aumentar su rentabilidad y mejorar la atención al cliente.
# Datasets
plans.csv información de los planes actuales(precio, minutos incluidos, GB incluidos, costo por extra)
users.csv información de los clientes (edad, ciudad, fecha de registro, plan, churn)
usage.csv detalle del uso real de los servicios (llamadas y mensajes)
# Etapas del analisis realizado
Cargar y explorar
Identificación de problemas de calidad de datos
Limpieza básica de datos
Summary statistics de uso por usuario
Visualización de distribuciones y outliers
Segmentación de clientes
Insight para Stakeholders
